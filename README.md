## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

*Note  
Since you are using firebase, you will need to adapt the following files to your personal settings.

- firebaseConfig.ts  
- GoogleService-info.plist  
- app.json
